# Payment Service
## Use SDKManRC
https://sdkman.io/usage/
### Set Java version
```
sdk env
```


## Build app

```
./gradlew clean build
```

## Run app
```
./gradlew bootRun
```
Or simply from Intellij. Run Application class (No build is needed).
